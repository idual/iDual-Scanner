# iDual Scanner

iDual Scanner ist eine iPhone- und iPad-App zum schnellen Scannen von Barcodes und zur Übertragung der Ergebnisse an einen Zielrechner im lokalen Netzwerk.

## Highlights

- Scan von gängigen 1D- und 2D-Codes (z. B. EAN, Code 128, Code 39, QR, DataMatrix)
- Übersichtliche Liste der erkannten Codes
- Manuelles Senden, optional auch automatisches Senden
- Empfänger im lokalen Netzwerk finden (Bonjour/mDNS)
- Einstellungen für Scan-Verhalten und unterstützte Barcode-Typen
